# Building Energy Consumption Dashboard

This project analyzes building energy usage data using SQL and Tableau.

## Tools Used
- MySQL
- SQL
- Tableau Public

## Project Features
- Average energy consumption by building type
- Temperature vs energy consumption analysis
- Temperature vs HVAC consumption analysis
- Interactive dashboard filter by building type

## Files
- energy_queries.sql
- query_1.sql
- query_2.sql
- query_3.sql
- Building Energy Consumption Dashboard.twbx
- dashboard_screenshot.png


## Key Insights
- Commercial buildings had the highest average energy consumption.
- Energy consumption increased as temperature increased.
- HVAC consumption was strongly related to temperature.
